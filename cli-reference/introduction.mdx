## âš™ï¸ Command Syntax

```bash
verisynth <input_csv> -o <output_dir> [options]
```

**Options:**

| Flag         | Description                                        | Default           |
| ------------ | -------------------------------------------------- | ----------------- |
| `--rows`     | Number of synthetic rows to generate               | 1000              |
| `--seed`     | Random seed for deterministic generation           | 42                |
| `--model`    | Synthesis model (`gaussian_copula`, `ctgan`, etc.) | `gaussian_copula` |
| `--proof`    | Output proof file path                             | `proof.json`      |
| `--no-proof` | Disable proof generation                           | Off               |
| `--quiet`    | Suppress console output                            | Off               |

Example:

```bash
verisynth data/finance.csv -o out/ --rows 50000 --seed 7 --model gaussian_copula
```
